# JianSan(Tri-dice) 兼三

三粒骰子酒令游戏。详细资料可参阅明代屠豳叟的《兼三图》，收录在明代陶挺《说郛续·卷三十九》中。

## 规则

- 胜采：浑色(111, 222, 333, 444, 555, 666)

    - 浑一、浑四、浑六赏 5 筹
    - 浑二、浑三、浑五赏 4 筹
    
- 余采(三粒骰子合计点数)

    - 四至六点，赏 2 筹
    - 七点八点，赏 1 筹
    - 九至十一点，赏 2 筹
    - 十二点至十四点，赏 3 筹
    - 十五点至十七点， 赏 4 筹
    
## 三粒骰子概率分布

- 甲甲甲： 6/216
- 甲甲乙： 90/216
- 甲乙丙： 120/216（顺子概率 24/216，杂点概率 96/216）
